# Text2Speech-Converter
Link:

A simple Python GUI application that converts **Text to Speech** and **Speech to Text** using Google's Speech APIs.
Built using **Python**, **Tkinter**, **gTTS**, and **SpeechRecognition**.



## 📌 Features

✅ Convert Text to Speech

✅ Convert Speech to Text

✅ User-friendly GUI

✅ Supports multiple languages

✅ Saves generated speech as MP3

✅ Displays recognized speech instantly



## 🖥️ Technologies Used

- Python
- Tkinter
- gTTS
- SpeechRecognition
- PyAudio


## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Speech-Text-Converter.git
```

Move inside project

```bash
cd Speech-Text-Converter
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```



## 🚀 How to Use

### Text to Speech

1. Enter text.
2. Enter language code.
3. Click **Convert Text to Speech**.
4. Audio is saved as **text.mp3**.



### Speech to Text

1. Enter recording duration.
2. Click **Convert Speech to Text**.
3. Speak through your microphone.
4. Recognized text will appear.
